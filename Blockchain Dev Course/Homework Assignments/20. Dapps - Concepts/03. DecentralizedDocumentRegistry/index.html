Document Registry

Welcome to the "Document Registry" DApp. This distributed app runs on the Ethereum blockchain network and uses a smart contract to manage the registry.
  • The documents are stored in IPFS and accessed via a hash. The registry keeps the hashes along with the publish date.
  • Contract owner can submit new documents to be stored on the blockchain.
  • Users can view documents in the registry.

Submit a Document

Contract owners can register (upload) new documents to the "Document Registry" smart contract on the Ethereum network.


View Documents

Blockchain users can get the document hash from the registry and retrieve the document from IPFS.
